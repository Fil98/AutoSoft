# AutoSoft
AutoSoft PowerShell in Windows 10

Список программ:
thunderbird 
git 
telegram 
flameshot
taskbarx 
winrar 
GoogleChrome
VisualStudioCode

Запуск скрипта:
1. Для выполнения сценариев PowerShell ввести команду: **Set-ExecutionPolicy RemoteSigned**
2. После запускаем скрипт: **.\AutoSoft.ps1**
3. Ждем окончания установки.
